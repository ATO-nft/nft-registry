# NFT Registry

Shows the current status of each registered NFT.

## Install

```js
npm i
```

## Test

```js
npx hardhat test
```

## Deploy

Create a `.env` on the model of `.env.example`:

```js
cp .env.example .env
```

Add your own keys in your `.env` file. 

Then deploy to Goerli:

```js
npm run deploy
```
